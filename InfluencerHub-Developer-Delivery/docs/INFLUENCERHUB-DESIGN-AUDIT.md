# InfluencerHub — Design Audit (قبل نظام التصميم الأصلي)

تدقيق الحالة السابقة (قبل هوية `ih-`) عبر البوابات، ونقاط التطوير المنفّذة.

## الحالة السابقة
- **اللون**: أخضر تيل `#0d8a6f` كهوية أساسية (إرث). خط Tajawal فقط.
- **المكوّنات**: `.btn/.card/.table/.badge/.field/.modal/.sidebar/.nav-link` عامة، متغيّر واحد `--brand`.
- **الشريط الجانبي**: فاتح بحالة نشطة خضراء ناعمة.
- **التجاوب**: الشريط الجانبي 250px ثابت — يضغط المحتوى على الجوال (تجربة غير تطبيقية) رغم عدم وجود تمرير أفقي على مستوى الوثيقة.
- **الشارات**: ألوان حالة متفرّقة بلا نظام موحّد.

## ملاحظات التطوير (منفّذة)
| العنصر | قبل | بعد |
|--------|-----|-----|
| اللون الأساسي | أخضر تيل | بنفسجي `#6252E5` |
| الخط | Tajawal | IBM Plex Sans Arabic + Inter |
| الأزرار | نمط واحد + primary/ghost | 5 أنماط + 4 أحجام + حالات + تركيز |
| الشارات | ألوان متفرّقة | نظام حالات موحّد `.ih-status-*` بنقطة |
| الجداول | قد تُمرّر الصفحة | تمرير داخلي، رأس ناعم |
| النماذج | 0.9rem (تكبير iOS) | 16px على الجوال، حلقة تركيز بنفسجية |
| الجوال | شريط جانبي يضغط المحتوى | Drawer + شريط علوي + عرض كامل |
| Tokens | متغيّر واحد | نظام `--ih-*` كامل + جسر توافق |

## البوابات المُعايَنة حيًّا
الوكالة `/app`، العميل `/client`، المبدع `/creator`، الشريك `/partner` — كلها ترث الهوية الجديدة عبر جسر التوافق دون تعديل ملفات الصفحات.
