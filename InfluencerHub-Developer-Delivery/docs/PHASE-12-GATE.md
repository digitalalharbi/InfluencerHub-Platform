# Phase 12 — التقارير والتحليلات (Analytics / Reports) — بوابة القبول

الحالة: **PHASE COMPLETE** (Feature Complete + مُتحقَّق بالمتصفّح + اختبارات خلفية). التالي: Phase 13.

## ما أُنجز
- **البنية** `app/Domain/Analytics/Services/AnalyticsService`: تجميعات حقيقية مقيّدة بالمستأجر عبر كل الوحدات (عملاء/مبدعون/طلبات/حملات/تعاونات/محتوى/عقود/مستحقات).
- **لوحة التقارير** `/app/reports` (فعّلت placeholder «التقارير»): بطاقات مؤشّرات رئيسية + رسوم أشرطة بسيطة (بلا مكتبة خارجية) لكل تصنيف حالة، مع إجماليات مالية بوحدات صغرى.
- **صدق البيانات**: كل الأرقام محسوبة لحظيًا من قاعدة بيانات المستأجر — لا أرقام تجريبية، لا mock.

## الأمان (اختبارات + حيًّا)
- عزل مستأجر تام: تجميعات مستأجر لا تتضمّن بيانات آخر (مؤكَّد باختبار).
- العرض متاح لأدوار العرض (viewer فأعلى).

## الاختبارات
- **خلفي: 335 إجمالًا**. `AnalyticsTest(4)`: التجميعات تعكس البيانات الفعلية، عزل المستأجر (لا تسرّب)، عرض الصفحة عبر HTTP، وصول دور العرض.
- **حيًّا**: اللوحة تعرض بيانات المراحل الفعلية (مستحقات مدفوعة 13,500، تعاون مقبول 1، محتوى في مراجعة العميل 1، حملة نشطة 1، طلب في الفرز 1، مبدعون حسب النوع).
