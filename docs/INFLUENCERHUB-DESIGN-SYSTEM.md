# InfluencerHub — Design System (ih-*)

هوية أصلية مستقلة. لا بادئة `sc-`، لا اسم/شعار/لون Smart Code، لا أخضر كهوية أساسية.

## المصدر
- `resources/css/tokens.css` — كل الـTokens (`--ih-*`) + جسر توافق يعيد تعيين المتغيّرات القديمة (`--brand` → بنفسجي) فترث كل الواجهات الهوية الجديدة دون تعديل ملفاتها.
- `resources/css/app.css` — Tailwind v4 + المكوّنات + الطبقة المتجاوبة (App Shell، جداول، نماذج).

## اللون
- **الرئيسي (بنفسجي)**: `--ih-primary` = `#6252E5` (600). التدرّج 50→950.
- **التكامل/الذكاء (سماوي)**: `--ih-accent` = `#06B6D4`.
- **الأسطح الداكنة (Ink)**: `#080D1A`→`#172238` (جاهزة للـApp Shell الداكن والـDark Mode).
- **المحايد**: رمادي 25→900.
- **الحالات**: success `#16A34A` (النجاح فقط — ليس هوية)، warning `#D97706`، danger `#DC2626`، info `#2563EB` — لكلٍّ soft + ink.
- استخدام البنفسجي: الإجراء الأساسي، التنقل النشط، حلقات التركيز، المؤشرات الرئيسية. السماوي: AI/التكاملات/الحيّ.

## الطباعة
- عربي: **IBM Plex Sans Arabic**، لاتيني: **Inter**، احتياطي Tajawal/النظام. مقياس بـ`clamp()` (display/page/section/card/body/sm/caption).

## المكوّنات (الحالية)
- الأزرار: primary/secondary/outline/ghost/danger + xs/sm/md/lg + block، حالات hover/active/disabled، حلقة تركيز.
- البطاقات/اللوحات، الجداول (رأس لاصق ناعم، تمرير داخلي `.ih-table-wrap`)، الشارات + **نظام حالات موحّد** (`.ih-status-*`)، النماذج (16px على الجوال لمنع تكبير iOS، حلقة تركيز بنفسجية)، النافذة/الـDrawer.

## App Shell المتجاوب
- سطح مكتب: شريط جانبي فاتح احترافي، الحالة النشطة بنفسجية ناعمة بحد داخلي.
- جوال (<768px): شريط علوي بزرّ قائمة (SVG)، الشريط الجانبي **Drawer جانبي من اليمين (RTL)** مع Scrim، محتوى بعرض كامل، أهداف لمس ≥44px، `100dvh`، `safe-area`.

## القواعد
- بلا تمرير أفقي عام (`overflow-x:hidden` + `max-width:100%` + تمرير الجداول داخليًا).
- Tokens مركزية — لا قيم لون مكرّرة داخل الصفحات.
- `prefers-reduced-motion` محترمة. حركة 140–240ms.
- Dark-mode readiness عبر `:root[data-theme="dark"]`.
