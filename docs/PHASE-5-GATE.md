# Phase 5 — بوابة العميل والوكالة الخارجية — بوابة القبول الكاملة

الحالة: **PHASE COMPLETE** (Feature Complete + مُتحقَّق بالمتصفّح + اختبارات خلفية + E2E). التالي: Phase 6.

> تسمية صادقة: مكتملة الميزات ومُختبَرة (وحدة + E2E) ومُعايَنة حيًّا. ليست "PRODUCTION READY" — قنوات البريد/SMS ما زالت `waiting_for_credentials`، والتصلّب الأمني الشامل + النشر ضمن مراحل لاحقة.

## ما أُنجز في Phase 5

### بوابة العميل (`/client/*`)
دخول + مبدّل عملاء، لوحة، ملف (مباشر + حقول قانونية عبر مراجعة)، ملف مالي، عناوين، مستندات خاصة (إصدارات/رؤية/تنزيل مُدقّق)، سير عمل العلامات (مسودة→إرسال→إصدارات + لافتة سبب)، فريق (دعوة/دور/حالة + حماية آخر مدير)، إشعارات (مركز + شارة + وصل بالأحداث)، إعدادات (تفضيلات/كلمة مرور/جلسات/2FA readiness).

### مراجعات الوكالة (`/app/*`)
- `/app/brand-reviews`: قبول العلامات بالأحداث (آلة حالة، سجل قرارات append-only).
- `/app/client-reviews`: طلبات التعديل القانوني (اعتماد يُطبّق/رفض) + مراجعة المستندات.

### الوكالة الخارجية والشريك
- `/app/partner-agencies`: قبول بالأحداث + دعوة أعضاء + روابط مُنطّقة (scoped) بعميل/علامة.
- `/partner/*`: بوابة شريك مستقلة، `EnsurePartnerMember` fail-closed (عضوية نشطة **و** وكالة معتمدة)، لوحة تعرض الروابط المُنطّقة فقط.
- `/partner/invite/{token}`: قبول دعوة عام مُحصّن (hash فقط، أحادي الاستخدام، Rate limited، إنشاء حساب + دخول).

### بنية تحتية
- `app/Domain/Communications`: نظام إشعارات محايد للمزوّد (in_app فوري، email/sms = waiting_for_credentials).
- `app/Domain/Partners`: الوكالات الخارجية + الأعضاء + الروابط المُنطّقة + آلة حالة القبول.

## الأمان (مُتحقَّق باختبارات + حيًّا)
- عزل مستأجر fail-closed (TenantContext + TenantScope + route-binding 404 عبر المستأجرين).
- IDOR-safe لكل مورد (مقيَّد بالعميل/الوكالة النشِطة).
- بوابات مستقلة: `EnsureClientMember` و`EnsurePartnerMember` (عضوية نشطة، والوكالة معتمدة).
- صلاحيات دقيقة (ClientPortalAbilities/ClientPolicy/ExternalAgencyPolicy).
- روابط الشريك مُنطّقة: لا وصول لغير المُصرَّح؛ الربط بعد الاعتماد فقط؛ العلامة تخص العميل.
- رموز الدعوة مُخزّنة sha256، أحادية الاستخدام، Rate limited.
- بلا أسرار ثابتة؛ إشعارات صادقة (لا تسليم وهمي).
