# InfluencerHub — Portal Auth Experience

تجربة دخول احترافية عالية التحويل: قسم تسويقي (Navy + بنفسجي/سماوي) + قسم دخول، بهوية أصلية.

## البنية
- `resources/views/layouts/auth.blade.php` — سقالة عمودين قابلة لإعادة الاستخدام (Sections: portal_tag/headline/sub/benefits/form_title/form_sub/form/portal_switch).
- `components/ih-logo.blade.php` — علامة أصلية (Hub يربط عقدتين) قابلة للاستبدال بـSVG لاحقًا.
- `components/ih-benefit.blade.php` — بطاقة ميزة بأيقونة SVG (نمط Lucide).
- CSS: `.ih-auth*` في `app.css`.

## البوابات
| البوابة | العنوان | CTA |
|---------|---------|-----|
| الوكالة `/login` | أدر عمليات المؤثرين والعملاء من منصّة واحدة | دخول إلى الوكالة + محوّل بوابات + طلب انضمام |
| العميل `/client/login` | تابع حملاتك وموافقاتك وتقاريرك بكل وضوح | دخول إلى بوابة العميل |
| المؤثر/صانع المحتوى `/creator/login` | أدر فرصك وتعاوناتك ومستحقاتك باحتراف | دخول + انضم الآن |
| الشريك `/partner/login` | أدر عملك المشترك مع الوكالة بكفاءة ووضوح | دخول إلى بوابة الشريك |

## الرسائل
احترافية، تنفيذية، سعودية/خليجية، بلا ادعاءات: «منظومة تشغيل متكاملة… بيئة موحّدة بدل الرسائل والملفات المتفرّقة». بطاقات القوة: إدارة موحّدة، أتمتة وتشغيل، تحليلات وتقارير، عقود ومدفوعات. عنصر ثقة: «بيئة تشغيل آمنة · صلاحيات منظّمة · سجل نشاط».

## الجوال
Mobile-first: **الفورم في الأعلى**، القسم التسويقي مختصر أسفله (order). أهداف لمس كبيرة، حقول 16px (لا تكبير iOS)، بلا تمرير أفقي، محوّل بوابات كـChips.
